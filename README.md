# Heimdall Luxury Fragrances - E-Commerce Website

A premium, fully responsive e-commerce website for Heimdall luxury fragrances, inspired by high-end niche fragrance brands.

## Features

- **Home Page**: Hero section with brand introduction and featured products
- **Products/Shop Page**: Complete product grid with detailed fragrance information
- **Categories Page**: Men's, Women's, and Unisex fragrance collections
- **About Page**: Brand story, mission, and values
- **Contact Page**: Contact form and business information
- **Shopping Cart**: Functional cart with local storage persistence
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop

## Files Included

- `index.html` - Home page
- `products.html` - Shop/Products page
- `categories.html` - Categories page
- `about.html` - About page
- `contact.html` - Contact page
- `style.css` - Complete styling
- `script.js` - JavaScript functionality
- `README.md` - This file

## Design Features

- Luxury color palette (deep blues, purples, gold accents)
- Premium typography (Playfair Display + Inter)
- Smooth animations and transitions
- Modern, minimalist aesthetic
- High-end brand presentation

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the site using the menu
3. Add products to cart (stored in browser localStorage)
4. All pages are fully functional

## Customization

All content is easily editable:
- **Colors**: Edit CSS variables in `style.css` (`:root` section)
- **Products**: Modify product cards in HTML files
- **Text**: All text content is in HTML files
- **Images**: Replace placeholder images with actual product photos

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- Product images use placeholder URLs - replace with actual images
- Contact form is UI-only (no backend integration)
- Cart uses localStorage (data persists in browser)
- No payment integration (buttons are UI-only)
